# 📊 לוח בקרה פיננסי - PWA

אפליקציית ניהול נכסים פיננסיים עם תמיכה מלאה ב-PWA (Progressive Web App)

## 🎯 מה כלול?

✅ **portfolio-dashboard.html** - האפליקציה המלאה
✅ **manifest.json** - הגדרות PWA
✅ **service-worker.js** - עבודה בלי אינטרנט
✅ **README.md** - ההוראות שאתה קורא עכשיו

## 🚀 איך מעלים? (3 דקות!)

### אופציה 1: GitHub Pages (מומלץ) ⭐

#### שלב 1: צור חשבון ב-GitHub
1. גש ל-https://github.com
2. לחץ על "Sign up" וצור חשבון חינם

#### שלב 2: צור Repository חדש
1. לחץ על הכפתור הירוק "New" 
2. שם: `my-portfolio` (או כל שם שתרצה)
3. סמן ✅ "Public"
4. לחץ "Create repository"

#### שלב 3: העלה קבצים
1. לחץ "uploading an existing file"
2. גרור את 3 הקבצים:
   - `portfolio-dashboard.html` ← **שנה שם ל-** `index.html`
   - `manifest.json`
   - `service-worker.js`
3. לחץ "Commit changes"

#### שלב 4: הפעל GitHub Pages
1. לחץ על "Settings" (גלגל שיניים למעלה)
2. בתפריט משמאל לחץ "Pages"
3. תחת "Source" בחר: **main** branch
4. לחץ "Save"

#### שלב 5: זהו! ✅
- אחרי 2-3 דקות האפליקציה תהיה חיה ב:
  `https://YOUR-USERNAME.github.io/my-portfolio`

---

### אופציה 2: Netlify (הכי קל!)

#### שלב 1: גש ל-Netlify
1. פתח https://www.netlify.com
2. לחץ "Sign up" והתחבר עם Google/GitHub

#### שלב 2: העלה קבצים
1. לחץ "Add new site" > "Deploy manually"
2. **שנה שם** `portfolio-dashboard.html` ל- `index.html`
3. גרור את כל 3 הקבצים לאזור:
   - `index.html`
   - `manifest.json`
   - `service-worker.js`

#### שלב 3: זהו! ✅
- תקבל קישור מיידית כמו: `https://random-name-123.netlify.app`
- אפשר לשנות את השם בהגדרות

---

### אופציה 3: Vercel

#### שלב 1: גש ל-Vercel
1. פתח https://vercel.com
2. לחץ "Sign up" והתחבר

#### שלב 2: העלה
1. לחץ "Add New" > "Project"
2. בחר "Import from file system"
3. **שנה שם** `portfolio-dashboard.html` ל- `index.html`
4. העלה את 3 הקבצים
5. לחץ "Deploy"

#### שלב 3: זהו! ✅
- תקבל קישור כמו: `https://my-portfolio.vercel.app`

---

## 📱 איך מתקינים כאפליקציה?

### באנדרואיד:
1. פתח את הקישור בכרום
2. לחץ על התפריט (3 נקודות)
3. בחר "הוסף למסך הבית" או "התקן אפליקציה"

### ב-iPhone/iPad:
1. פתח את הקישור בספארי
2. לחץ על כפתור השיתוף (ריבוע עם חץ)
3. גלול למטה ובחר "הוסף למסך הבית"

## 💡 טיפים

### רוצה דומיין משלך?
- ב-Netlify/Vercel אפשר להוסיף דומיין חינם
- אפשר לקנות דומיין ב-Namecheap/GoDaddy (~$10/שנה)

### רוצה לעדכן?
- פשוט העלה שוב את הקבצים
- השינויים יעלו אוטומטית

### בעיות?
- ודא ששינית את השם ל-`index.html`
- ודא שכל 3 הקבצים באותה תיקייה
- נסה למחוק cache בדפדפן (Ctrl+Shift+R)
